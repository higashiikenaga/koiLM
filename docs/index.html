koiLM

完全ローカル動作の恋愛・会話AIアプリ
外部通信なし、あなたの端末だけで動きます

ダウンロード (Releases) GitHub

特徴

  • 完全ローカルで動作するローカルLLM搭載の恋愛チャットアプリ(通信なし・プライバシー安心)
  • Windows版(ポータブルexe)とWebUI版(PCで起動し、同じWi-Fi内のスマホからも遊べます)
  • 日本語 / English 切り替え対応(言語ごとに専用モデルを使用)
  • 好感度システム(既読タイミング・返信間隔の演出)
  • キャラクター作成時の1枚絵自動生成
  • 年齢確認・表現レベル設定(通常表現 / SFW / NSFW)

導入方法

  1. Releases から、お使いの環境に合ったパッケージをダウンロード
  2. 展開したフォルダの構成に沿って、モデルファイルを models/ に配置
  3. Windows版は koiLM.exe をダブルクリック、WebUI版は npm start で起動
  4. WebUI版はスマホから http://(PCのIPアドレス):3939 にアクセスすると同じWi-Fi内から遊べます

高品質モデル(7B)について

デフォルトの軽量モデルに加えて、より高品質な7Bクラスのモデルも利用できます。 ファイルサイズが大きいため、Hugging Faceで別途配布しています。詳細は READMEのダウンロード欄を参照してください。
